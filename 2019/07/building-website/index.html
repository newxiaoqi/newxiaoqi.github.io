建站的心路历程

七月 6, 2019 发布在 日志

这是一篇关于搭建本站踩到的一些坑的记录和心路历程。

花了将将一周时间把这个个人博客搭建起来了，成本为：域名购买费（一年），9元。大佬排错费，5元。共计：14元。

花了两天时间在Hexo和Hugo之间做出抉择，最终选择了用Hugo来搭建，放到github上。自己不是码农所以也算第一次正式学习如何使用github，发现确实有些复杂，不过还好，跟着网上的教程一步一步也就做过来了。这期间经历了N次重新搭建网站的事故，包括一次整体删库，索性的是自己还耐着性子继续做下来了，最后呈现出这么一个效果。

这期间踩了不少坑，其中最大的坑是chrome内核的浏览器不显示主题，自己排查了许久也不得要领，最终还是靠大佬一句话我才明白问题所在。网站的CSS没有ssh，被谷歌限制了，只要在baseURL中把github.io的域名改成https://自己购买的域名就好了。

关于域名解析： 购买域名后在github中放入CNAME,在域名解析中加入CNAME和A记录就可以了，剩下的就是等待DNS的同步。不过不得不说阿里云的DNS解析速度太慢了，而且时好时坏，有点坑了。

关于博客更新计划： 主要以个人学习机器学习及深度学习的笔记为主，学科论文等兴趣为辅，其他的东西能皮一点就皮一点，有时候还会发发福利。

这是第一篇博文，就当是一个好的开始，今后要加油了。
